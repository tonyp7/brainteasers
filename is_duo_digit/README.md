# is_duo_digit

Write a function is_duo_digit that takes as a parameter an integer. The function should return 'y' if the number is made of no more than two unique digits.

For example:

is_duo_digit(12) => 'y'
is_duo_digit(-3333) => 'y'
is_duo_digit(102) => 'n'
is_duo_digit(789) => 'n'
is_duo_digit(7887) => 'y'
